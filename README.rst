python-template
===============

* Uses Poetry as the python virtual environment
Insallation instructions: https://python-poetry.org/docs/


To run the code, there are 2 options A and B

Option A:
------------
Activating the venv  
`poetry shell`

de-activating the venv:  
`exit`  
    
Option B:
------------
'poetry run my_python_app.py'
